# `<Derive>` and `derive()` Demo

Interactive demo of General Translation's `<Derive>` component and `derive()` string function for grammatical agreement across languages.

**[Live Demo →](https://static-demo.generaltranslation.dev)**

## Features

- 🌍 **5 languages** — English (default), Spanish, French, Japanese, and Chinese
- 🔄 **Locale switcher** — Built-in `<LocaleSelector>` component
- 🧩 **`<Derive>` component** — Generates separate translations per permutation for grammatical agreement
- 🔤 **`derive()` function** — String equivalent for template literal translations
- ⚡ **Next.js 16** — App Router with Turbopack

## Quick Start

```bash
git clone https://github.com/gt-examples/static-demo.git
cd static-demo
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and toggle the options to see how `<Derive>` generates separate translations for each permutation.
